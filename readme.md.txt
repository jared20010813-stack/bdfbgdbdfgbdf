# YourDoge Unblocker

Custom Scramjet-powered unblocker with retro Doge theme.

## Features
- Clean Doge design
- Dedicated Games page with 12+ popular titles
- Scramjet proxy support
- Fully static (GitHub Pages ready)

## Deployment Steps
1. Create a new repository on GitHub
2. Upload these files:
   - `index.html`
   - `games.html`
   - `sw.js`
   - `main.svg` (your logo)
   - `baremux/` folder (copy from PeteZah repo)
3. Go to **Settings → Pages → Deploy from main branch → Save**
4. Your site will be live at `https://yourusername.github.io/repo-name`
